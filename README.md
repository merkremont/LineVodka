# LineVodka
_May the Vodka be with you..._

Tutorial
------
Recomended for python 2.7

Using git :

    $ git clone http://github.com/merkremont/LineVodka

Python packages :

    $ pip install thrift==0.9.3
    $ pip install rsa
    $ pip install requests

Run Vodka :

    $ python vodkabot.py
